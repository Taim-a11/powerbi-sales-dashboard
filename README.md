# 📊 Executive Sales Dashboard | Power BI Portfolio Project

## 📌 Project Overview

This project is an interactive Sales Dashboard built using Microsoft Power BI and the Superstore Sales dataset. It provides business insights into sales performance, profitability, customer behavior, product performance, and regional trends through interactive dashboards.

The goal of this project was to practice Power BI development by applying data cleaning, data modeling, DAX calculations, and dashboard design techniques.

# 📈 Dashboard Pages

## 1️⃣ Executive Dashboard

### Features

- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Monthly Sales Trend
- Sales by Region
- Sales by Category
- Profit by Category
- Interactive Slicers (Region, Category, Year)

---

## 2️⃣ Product Analysis Dashboard

### Features

- Top 10 Products by Sales
- Top 10 Products by Profit
- Sales by Sub-Category
- Profit by Sub-Category
- Sales by Ship Mode
- Sales vs Profit Scatter Chart

---

## 3️⃣ Customer Analysis Dashboard

### Features

- Top 10 Customers by Sales
- Top 10 Customers by Profit
- Sales by State
- Sales by Segment

---

# 📊 DAX Measures

```DAX
Total Sales = SUM('Sales Data'[Sales])

Total Profit = SUM('Sales Data'[Profit])

Total Orders = DISTINCTCOUNT('Sales Data'[Order ID])
```
